# 🧠 MindScape AI Coach

> Your personal AI-powered guide for emotional insight and smart life planning

---

## 📌 Overview
MindScape AI Coach is an intelligent application that analyzes user inputs such as age and emotional state to generate personalized roadmaps and guidance. It aims to support self-improvement, mental clarity, and better decision-making using AI.

---

## ✨ Features
✅ Personalized AI-generated roadmaps  
✅ Emotion-aware recommendations  
✅ Clean and simple user interaction  
✅ Scalable design for future AI integration  

---

## 🛠️ Tech Stack
- Python 🐍  
- Machine Learning Concepts 🤖  
- Flask (Backend) 🌐  
- Environment Variables (.env) 🔐
